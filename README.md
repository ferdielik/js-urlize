# js-urlize
Filter to create better seo urls and slugs with angularjs or javascript
Note: only support turkish characters

### angular usage ###
##### $filter('urlize')('Örnek Kullanım İçin') 
result -> "ornek-kullanim-icin"


### javascript usage ###
##### urlize('Örnek Kullanım İçin')
result -> "ornek-kullanim-icin"
